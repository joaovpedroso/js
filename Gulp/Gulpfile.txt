<h1>Instalação</h1>

Iniciar gulp no diretório do projeto
*npm install --save-dev gulp

Listar os plugins instalados
*npm ls --depth=0

Instalar Plugins
*Acessar site NPM: npmjs.com
*Pesquisar pelo plugin: gulp-NOME DO PLUGIN
*Executar: NPM Init 
*Executar o npm install do plugin

Criar tasks (Ações)
*gulp.task('nome', function(){});

Rodar ação do Gulp
*gulp 'nome da ação'

Observador Watch
*gulp watch



<h1> Utilização  </h1>
1?Instalar o node na máquina - nodejs.org
2?Instalar Gulp na Raiz do Projeto - gulpjs.com
3?Criar um arquivo gulpfile.js
4?Instalar os plugins - npmjs.com
5?Rodar o Watch - Observador
